What are Color Palettes ?

A color palette is a combination of colors used by UI designers when designing an interface which helps to maintain consistency, and make your user interface aesthetically pleasing and enjoyable to use.

Types of Color Palettes

There are several types of color palettes that can be used in your design, but broadly there are four main types.

1. Monochromatic Color Palettes

A monochromatic color palette is a color palette in which a single color tint is used as the basis for all shades and hues found within the image. The shade of color is varied by changes made to the saturation and/or brightness of the base color.

2. Analogous Color Palettes

An analogous color palette involves grouping of colors that neighbor each on the color wheel i.e. colors that are positioned next to each other on the color wheel. For instance, if you choose the color blue, you other blue shades around it.

3. Complementary Color Palettes

A complementary color palette is composed by using two colors which are opposite to each other on the color wheel. This is the particularly contrasting of all color schemes which attracts the most attention.

4. Triadic Color Palettes

A triadic color scheme is comprised of three colors evenly spaced on the color wheel. The two most basic triadic palettes are the primary colors red, blue, and yellow, and the secondary hues orange, purple, and green.
